Validate some naming conventions for unit tests, according to BCP bank rules. Currently, observing:
  • @DisplayName property must exist for every test
  • @DisplayName string must start with uppercase
  • @DisplayName string must not have spaces at the end or start of the string
  • Method name must contain the word 'When'
  • Method name must start with lowercase
  • Method name must correspond to the display name, but with spaces

Warning: This plugin was made to work exclusively with the codebase of the BCP bank of Peru.
